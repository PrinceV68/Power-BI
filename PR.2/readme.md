# 📊 Sales & Customer Intelligence Dashboard

An **Interactive Business Intelligence Dashboard** built with **Microsoft Power BI** for advanced sales, customer, and business performance analysis.

---

# 📌 Overview

The **Sales & Customer Intelligence Dashboard** is an end-to-end **Business Intelligence project** developed using **Microsoft Power BI**.

This dashboard transforms raw business data into actionable insights using **data modeling, advanced DAX calculations, KPI tracking, and interactive visualizations**.

The project helps organizations analyze:

* Sales Performance
* Customer Behavior
* Product Performance
* Regional Trends
* Return Analysis
* Business Growth Patterns

# 📂 Dataset

The project uses multiple interconnected tables designed using **Star Schema Architecture**.

| Table        | Description                  |
| ------------ | ---------------------------- |
| Sales_Fact   | Stores Sales Transactions    |
| Customer_Dim | Customer Information         |
| Product_Dim  | Product Details              |
| Region_Dim   | Regional Information         |
| Date_Dim     | Calendar & Time Intelligence |
| Returns_Fact | Return Transaction Records   |

---

# 🏗️ Data Model Architecture

```text id="schema100"
                Date_Dim
                    │
                    │
Product_Dim ─ Sales_Fact ─ Customer_Dim
                    │
                    │
             Returns_Fact
                    │
                    │
               Region_Dim
```

The dashboard follows **Star Schema Modeling** for better performance and scalable reporting.
